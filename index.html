To install the packages:

  1. Download and install JKI VI Package Manager
  2. Subscribe to the VI Package Repository at https://levylabpitt.github.io
  3. Check repositories for updates
  4. Install "General AFM Lithography" or "Lockin-Multichannel" from the LevyLab Packages repository.
    All required dependencies should be installed automatically.
